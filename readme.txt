杂志信息采集检索系统 使用说明


运行软件：
双击 run.exe 启动软件服务。
然后可以使用浏览器访问 http://127.0.0.1:8000/ 或者 http://(你的IP地址):8000/ 看到系统主界面


数据采集：
点击进入“数据采集”页，在文本框中填入你要获取的杂志页面的 url 地址，每行一个地址。
点击“开始采集”按钮。
软件会在后台针对输入的数据进行逐个采集，原则上，没有采集的过地址第一次采集会比较久。
之前采集过的地址再次采集时会自动忽略那些已经采集过的分期，只采集最新更新的分期。


数据管理：
点击进入“数据管理”页，可以看到所有采集下来的文章信息。
可以对数据进行修改，点击修改按钮，在下面出现六个文本框，分别对象数据记录的六个字段，改成你需要的数据后，点击确定按钮，完成修改。
